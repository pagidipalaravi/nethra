Why we have to use Conditional Statements?
	While we are writing a code, we want to perform actions for different conditions.
	So, conditional statements are used to perform different actions based on different conditions.
What are the conditional statements we have in Javascript?
	We have the following Conditional Statements:
		if - if statement is used to specify a block of code will executed, if a specified condition is true.
		else - else statement is used to specify a block of code to be executed, if a specified condition is false.
		else if - else if statement is used to test the new condition, if the condition is false.
		switch - It is used to specify many alternative blocks of code to be executed.
What is if statement?
	if statement is used to specify a block of code will executed, if a specified condition is true.
	if should be in lowercase, Uppercase letters (IF) will generate a Javascript error.
What is else Statement?
	else statement is used to specify a block of code to be executed, if a specified condition is false.
What is else if statement?
	else if statement is used to specify a new condition , if the condition if false.
