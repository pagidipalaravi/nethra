How to view HTML source code?
	Right-click in an HTML page.
	select "View Page Source" (in Chrome)
	select "View Source" (in Edge)
	or similar in other browsers.
	This will open a window containing the HTML source code of the page.
How to inspect an HTML element?
	Right-click on an element (or a blank area)
	choose "Inspect" or "Inspect Element" to see what elements are made up of 
	You can also edit the HTML or CSS on-the-fly in the Elements or Styles panel that opens.
